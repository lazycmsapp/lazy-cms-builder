# Builder Library

The Library lets you save containers and columns as reusable templates. Unlike Global Sections, library items are **independent copies** — editing one insert doesn't affect others.

## Saving to Library

### Save a Container

1. Hover over a container
2. Click the **📚 Library** icon on the container toolbar
3. Give it a name (e.g., "Hero Banner Blue")
4. Click **Save**

### Save a Column

1. Hover over a column
2. Click the **📚 Library** icon on the column toolbar
3. Give it a name (e.g., "Feature Card")
4. Click **Save**

## Using Library Items

1. Click **+ Add Container** to open the Column Select modal
2. Click the **Container Library** tab (or **Column Library** when adding columns)
3. Find your saved item
4. Click **Add Container** / **Add Column**

The item is inserted as an independent copy — you can edit it freely without affecting other pages.

## Managing the Library

From the Column Select modal:
- **Delete** any saved item with the 🗑️ button
- Library items are shared across all pages

## Library vs Global Sections

| Feature | Library | Global Section |
|---|---|---|
| Edit propagates everywhere | ❌ No | ✅ Yes |
| Edit per-page | ✅ Yes | ❌ No |
| Best for | Reusing a design as a starting point | Consistent content across pages (header CTA, promo banner) |
