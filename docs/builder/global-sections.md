# Global Sections

A Global Section is a container saved once and reused across multiple pages. When you edit the Global Section, every page that uses it updates automatically.

## Creating a Global Section

1. Open any page in the Lazy Builder
2. Build your container (e.g., a newsletter signup banner)
3. Right-click the container toolbar
4. Select **Save as Global Section**
5. Give it a name (e.g., "Newsletter Banner")

The section is saved and available site-wide.

## Inserting a Global Section

1. Open any page in the Lazy Builder
2. Click **+ Add Container** (or click between containers)
3. In the **Column Select** modal, click the **Global Sections** tab
4. Click **Insert** on the section you want

The container appears on the page, linked to the Global Section.

## Editing a Global Section

**Option 1 — Edit in-place:**
Click the Global Section container in the builder and edit its elements normally. Changes save to the global section and propagate everywhere.

**Option 2 — From the Sections Manager:**
1. Go to **Admin → Appearance → Builder Sections**
2. Find your section and click **Edit**
3. Opens the builder for that specific section

## Deleting a Global Section

1. Go to **Admin → Appearance → Builder Sections**
2. Click **Delete**

::: warning
Deleting a Global Section removes it from all pages that were using it.
:::

## Global Section vs Library

| | Global Section | Library |
|---|---|---|
| Synced across pages | ✅ Yes — edit once, update everywhere | ❌ No — each insert is independent |
| Use case | Repeated elements (header CTA, newsletter banner, sale strip) | Save a design to reuse as a starting point |
| Editable per-page | ❌ No | ✅ Yes |
